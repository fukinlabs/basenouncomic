# Public Assets

Place your Farcaster manifest images in this directory:

- `icon.png` - App icon (recommended: 512x512px)
- `splash.png` - Splash screen image
- `hero.png` - Hero image for app display
- `screenshot-portrait.png` - Portrait screenshot of your app
- `og-image.png` - Open Graph image for social sharing (recommended: 1200x630px)

Update the URLs in `minikit.config.ts` after adding your images.
